当在模板中直接调用方法，而不使用 computed 时，你会发现，你改变一个无关紧要的状态，方法都会重新执行。

```vue
<template>
  <div>
    {{ getText() }}
    <span>{{ a }}</span>
  </div>
</template>

<!-->改变 a 的值 getText() 方法会重新执行<-->
```
